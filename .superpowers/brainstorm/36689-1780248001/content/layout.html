A

Slider Track Approach (Recommended)

All branches are in a single horizontal row. We move the track left/right. To prevent seeing Branch 2 when going from 1 to 3, we temporarily hide or shrink Branch 2 during the animation. This matches your idea of a sliding track.

B

Absolute Stack Approach

Each branch is stacked on top of each other. When animating from 1 to 3, Branch 3 slides in from the right, and Branch 1 slides out to the left. Branch 2 is simply not involved at all, ensuring you never see it mid-transition.
