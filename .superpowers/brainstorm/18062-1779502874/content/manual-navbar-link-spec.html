Navbar Link: Manual Spec Sheet

Use this specification sheet to build the Navbar Link component manually in the Webflow Designer.

1. Component Element Hierarchy
[ListItem] (Class: nav-bar__big-li)
 ├── [Link Block] (Class: nav-bar__big-a, Attributes: data-hover="")
 │    ├── [Text Block] (Class: nav-bar__big-span, Attributes: data-underline-link="")
 │    ├── [Text Block] (Class: nav-bar__big-span-number)
 │    └── [Div Block] (Class: nav-bar__a-tag)
 │         └── [Div Block] (Class: tag)
 │              ├── [Div Block] (Class: button-bg)
 │              └── [Text Block] (Class: eyebrow is--relative)
 └── [Div Block] (Class: line is--nav-transparent)

3. Underline Hover Custom CSS (Add to page head)
[data-underline-link] {
  position: relative;
  cursor: pointer;
}
[data-underline-link]::before {
  content: "";
  position: absolute;
  bottom: -0.175em;
  left: 0;
  width: 100%;
  height: calc(0.0625em / 3);
  background-color: currentColor;
  transition: transform 0.6s cubic-bezier(0.625, 0.05, 0, 1);
  transform-origin: right;
  transform: scaleX(0) rotate(0.001deg);
}
[data-hover]:hover [data-underline-link]::before {
  transform-origin: left;
  transform: scaleX(1) rotate(0.001deg);
}
